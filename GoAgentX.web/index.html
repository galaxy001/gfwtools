GoAgentX


GoAgentX 是一个在 Mac OS X 下使用代理服务的图形界面控制软件，方便一般用户在 Mac OS X 上配置和使用 goagent、west-chamber-season-3、SSH、stunnel 以及 shadowsocks。

下载 GoAgentX 下载源代码 View on GitHub

功能介绍

  • 集成 goagent，west-chamber-season-3，SSH，stunnel，shadowsocks
  • 支持部署 goagent 服务端到 Google App Engine
  • 自动修改系统代理设置全局代理或使用 PAC 模式
  • 图形化界面设置客户端连接参数
  • 菜单栏图标，直接控制连接状态
  • 菜单栏快速切换服务以及是否使用 PAC
  • 用户登录时自动启动代理服务
  • 支持使用 Apple Script 控制 GoAgnetX

系统要求

  • Mac OS X 10.6 及以上版本系统
  • 支持 64 位的 Intel CPU

常见问题

  • 如何避免随系统启动时需要授权
  • 如何使用 goagent
  • 如何使用 Apple Script 控制 GoAgentX

如何编译

获取代码：

$ git clone https://github.com/ohdarling/GoAgentX
$ git submodule init
$ git submodule update

然后打开 Xcode 项目 GoAgentX.xcodeproj 进行编译即可。

如何提问题

进入 https://github.com/ohdarling/GoAgentX/issues/new 页面填写需求信息或 Bug 即可。
